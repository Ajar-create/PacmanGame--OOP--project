# PacmanGame--OOP--project

 Description:
This is a simple console-based Pacman game developed in Java using Object-Oriented Programming concepts.
The player moves inside a grid to collect food while avoiding the ghost.

 Concepts Used:
. Classes and Objects
. Constructors
. Methods
. 2D Arrays
. Loops and Conditions

 Game Logic:
.The game works on a grid system where:
.Pacman moves using keyboard input (U, D, L, R)
.Each time Pacman collects food, the score increases by 20 points
.If Pacman hits a wall, the movement is not allowed
.If the ghost catches Pacman, the game immediately ends (Game Over)
.If all food items are collected, the player wins

 Features:
. Player movement (U, D, L, R)
. Ghost enemy
. Food collection system
. Score tracking
. Win and Game Over conditions

 How to Run:
1. Open project in any Java IDE
2. Compile all ".java" files
3. Run "PacmanGame.java"
